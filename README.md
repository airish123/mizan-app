
Mizan is a minimalist, Zen-inspired dashboard I built to keep my life balanced while grinding through A-Level Physics, Math, and CS. It’s designed to be a "single-page sanctuary" for Muslim students who need to stay productive without losing their spiritual focus.

## Tech Stack
- **React.js** (Hooks for the hidden-word logic)
- **Tailwind CSS** (Extreme minimalism/Soft dark mode)
- **Vibe Coding** (Developed using natural language architecture)

## How to Run This
If you're bored too and want to see how it works:
1. `git clone https://github.com/airish123/mizan.git`
2. `npm install`
3. `npm run dev`
